# NULL

## 개념

**NULL** : 데이터를 저장할 공간은 존재하지만, 값이 아직 입력되지 않은 상태

### 특징

비교연산/산술연산에서 정상 작동 x. -> [연산결과 모두 NULL.](함수#null함수)

집계 함수에서 제외됨.
숫자 0이나 빈 문자열''과 명백히 다르다.

## 데이터 모델에서의 표현법

### IE 표기법에서 NULL 표현

별도 표기법 없음.

속성이 입력된 박스 안에 선을 기준으로 주식별자와 일반속성 구분.

주식별자는 NULL 비허용

### Barker 표기법에서 NULL 표현

속성이 입력된 박스 안에 속성 옆에 기호를 표시.

\* : NULL 비허용(NOT NULL)

o : NULL 허용.

\# : 주식별자(PK)표현.
