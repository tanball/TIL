## 데이터시각화


### 데이터시각화 – matplotlib 활용 (이론+실습)


**데이터 시각화란?**

데이터의 분포나 특징을 확인하기 위해 차트 형태로 나타내는 일련의 과정

방대한 양의 데이터를 빠르게 보고 유의미한 특성을 찾아낼 수 있다.

데이터의 추세, 흐름, 특징 등을 한눈에 파악하고 논리적으로 설명하기 용이하다.


**Matplotlib 라이브러리**

파이썬에서 데이터를 차트나 플롯(Plot)으로 그려주는 가장 많이 사용되는 데이터 시각화(Data Visualization) 패키지


차트/플롯 종류

> line plot  : 
>
> 선그래프 라고도 부른다
> ```
> plt.plot(data)
> ```
>수치의 변화를 선으로 이어서 그려줌
>
>시간에 따른 데이터의 변화 추세를 볼 때 사용


> 바 차트 : 
>
> 막대그래프 라고도 부름
> ```
> plt.bar(x,height)
> ```
> 범주형 데이터의 분포를 그래프로 보여줌
>
> 보통 가로, 세로, 누적, 그룹화 된 막대그래프 사용


> 파이 차트 : 
>
> 원그래프 라고도 부름
>
> 강의 중 설명 생략
>
> 전체 대비 각 범주의 구성비 보여줌.
>
> 각 범주의 상대적 비중, 어떤 항목이 지배적인지 등


> 히스토그램 : 
>```
> plt.hist(x)
> ```
> 수치형 데이터의 분포 표현
>
> 빈도, 빈도밀도, 확률 등의 분포를 그릴 때 사용
>
> 구간 값(bins) 별도 지정 없다면 10개 구간으로 나누어 그림.


> box plot : 
> ```
> plt.boxplot(x)
> ```
> 수치적 자료를 표현하는 그래프
> 
> 통계량인 5가지 요약 수치(lower fence, q1, q2, q3, upper fence)를 가지고 그린다.


>scatter plot : 
>
>산점도
> ```
> plt.scatter(x, y)
> ```
>
> 두 값 간의 관계 표현에 사용
>
> 데이터 분포 표현에 가장 많이 쓰는 것.
> 
> x, y값을 지정해야 함.


차트 그리기 방법

*향후 matplotlib 약칭 plt

1.	메소드 : plt.figure()

    동작  :시각화 그림을 표시할 영역 지정

2.	메소드 : plt.plot() *혹은 각 플롯 별 함수

    동작 : 시각화 차트 및 값 지정

3.	메소드 : plt.show()

    동작 : 시각화 출력

2단계에서는 plot외에 다른 플롯 종류에 따른 다른 함수를 사용할 수도 있다



### 데이터시각화 – seaborn 활용 (이론+실습)


**Seaborn 라이브러리**

matplotlib 기반으로, 다양한 색상 테마와 통계용 차트 등의 기능을 추가한 시각화 패키지

-> 기본적 시각화 기능은 matplotlib 패키지에 의존



**matplotlib과 seaborn 비교**

|   | matplotlib |  seaborn |
| --- | --- | --- |
| 기능 | 기본적인 시각적 개체 만들기에 사용	| 주로 통계 그래프 만들기에 사용 |
| 표시 | 기본적이고 단순한 디스플레이 | 시각적인 요소가 가미된 디스플레이 |
| 사용방법 | numpy와 pandas만으로 다양한 시각화 가능 | numpy와 pandas 외에도 matplotlib도 사용해야 함. |
| 복잡성 | 복잡하고 긴 구문	| 쉽고 간단한 구문 |
| 다중성 | 여러 개의 시각화를 한번에 열 수 있음	| 메모리 부족 문제로 다중 시각화하는데 실패할 수 있음 |
| 유연성 | 강력하고 유연하며 사용자 기능 정의가 쉬움 | 비교적 덜 유연하고 사용자 기능 정의가 어려움 |





**seaborn에서 제공하는 차트들**

* 앞으로 코드에서 seaborn 은 약칭인 sns로 사용.

> **Scatter Plot(산점도)**
>```
> sns.scatterplot(x, y)
>```
> 두 값이 어떤 관계에 있는지 분포 확인 가능
>
> x,y값 지정해야 함.
>
> hue 파라미터 : 색상값 구분.

> **Bar Plot(막대그래프)**
>```
> sns.barplot(x, y)
>```
> x인자에는 x축에 표시할 범주형 변수의 값 입력
>
> y인자에는 각 범주에 해당하는 통계값 입력
>
> 범주별 y값에 대한 분포를 통해 비교 가능

> **Lm Plot**
>```
> sns.lmplot(x, y)
>```
> 산점도에 회귀선 그려줌
>
> 산점도에 대한 회귀 추세 알 수 있다.

> **Count Plot**
> ```
> sns.Countplot(x)
> ```
> 항목 별 개수 확인에 사용
>
> 연속형/범주형 변수 둘다 가능.
>
> 디자인적으로는 barplot과 매우 유사. 단, 통계값이 아니라 개수 표현.

> **BoxPlot**
>```
> sns.boxplot(x, y)
> ```
> 수치적 자료를 표현하는 그래프
>
> 수치적 자료에서 나오는 통계량인 5가지 요약 수치로 그래프 만듦
>
> *5가지 요약 수치 : Lower Fence, 제1사분위값, 제2사분위값, 제3사분위값, Upper Fence

> **Violin Plot**
>```
> sns.violinplot(x, y)
> ```
> 박스플롯과 유사하나 밀집도를 함께 볼 수 있어 데이터 탐색에 유용하게 사용 가능.

> Joint Plot
> ```
> sns.jointplot(x, y)
> ```
> 산점도와 히스토그램을 한번에 보여주는 그래프
> 
> 데이터의 분포와 상관관계를 한 번에 볼 수 있어 데이터 탐색에 장점이 있음.

> Heat Map
> ```
> sns.heatmap(data)
> ```
> 데이터 배열을 색상으로 표현하는 그래프
>
> 두 카테고리 값에 대한 값 변화를 한 눈에 알기 쉬워 두 값 또는 각 feature간 상관관계를 나타낼 때 주로 사용함.
