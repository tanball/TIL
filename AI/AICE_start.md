## Python Basic


**변수**

Variable. 변하는 수치. <->상수

하나의 방(주소)안에 우리가 기억해야 할 것을 저장.

> A = 100 #A라는 방에 100이라는 값(데이터)저장.

변수 할당 : 변수에 값을 지정. “=”을 사용하여 지정.

* 변수명 B1과 문자열 ‘B1’의 구분에 주의


**리스트(list)**

하나의 변수에 여러 개 값을 저장, 접근, 수정

선언

> num = [1, 2, 3, 4, 5] --> 대괄호 이용

> num = list(1, 2, 3, 4, 5)

> num = list(range(1, 11))

> num = list(range(1, 11, 2)) * range(시작값, 종료값, 간격) 간격은 디폴트1(생략가능)

리스트 함수(listname.TAB으로 확인)를 사용해 원소에 접근, 수정, 삭제 가능.

인덱스 : 리스트의 각 요소 위치에 접근. 첫번째 요소의 인덱스는 0. 또한 음수(-)표현을 통해 역순 접근도 가능하다.

> Num[0] 으로 첫번째 값에 접근

> Num[-1] 으로 마지막 값에 접근

리스트 내 원소의 자료형은 제한이 없다.(또한 통일될 필요도 없다.)


**파이썬 딕셔너리(Dictionary)**

하나의 변수에 여러 값을 저장, 접근, 수정, 삭제

Key – Value 구성. -> 값이 많아도 key 만 알면 바로 접속 가능

선언

> DIC = { k1 : v1, k2 : v2 }

> DIC2 = dict()

> DIC2[k] = v

>> dict = {‘name’: ‘pey’, ‘phone’: ‘010-1234-1234’, ‘age’: 25 }

>> dict[‘name’]

>> 출력 : ‘pey’

* 키의 값은 가변해서는 안된다. -> 리스트, 변수를 키로 가질 수 없음. 즉, 튜플, 문자열, 숫자 등을 키로 사용.

* list와 마찬가지로 한 딕셔너리 내에 여러 자료형 원소 저장 가능.


**조건문**

if문 : 조건절이 참이면 실행하고, 틀리면 실행하지 않는다.

문법

> if 조건문 :

>     실행문

> else : 

>     실행문2

이와 같은 형식으로 작성.


**반복문**


for문 : 반복적으로 문장을 수행한다

문법

> for 변수 in 리스트(또는 문자열):

>     수행할 문장 1

>     수행할 문장 2

> for i in [1, 2, 3, 4, 5]:

print(‘숫자를 {}을 카운트한다’.format(i))

> 익숙한 사용법은 for i in range(1,6):

파이썬의 range(시작값, 종료값) 함수를 이용해 연속된 정수 생성

단, 종료값을 포함하지 않음.(1~10 필요하면 1, 11로 입력해야)
